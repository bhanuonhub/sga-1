Observations:
- The workspace structure was created with separate directories for docs, src, and logs.
- Permissions were tightened so the main directory is accessible to the owner and group, while files are readable only by the owner.
- The default umask value is 0022, which helps create files with standard owner-read/write and group/other-read permissions.
- These permissions protect project data by limiting access to authorized users and reducing accidental modification.
